数据使用方法
---

```bash
npm install @province-city-china/utils --save-dev
```

```js
const { data, province, city, area, town } = require('province-city-china/utils');
```

- `data` - 总数据(省/地/县/乡)
- `province` - 省级(省/直辖市/特别行政区)
- `city` - 地级(城市)
- `area` - 县级(区县)
- `town` - 乡级(乡镇/街)
